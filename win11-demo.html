Usage Notes

This component library is inspired by Windows 11 design language but adapted for web use. Key features include:

  • Glassmorphism: Use backdrop-filter for blur effects on cards and panels
  • Rounded Corners: 8-12px border radius for most elements
  • Subtle Shadows: Layered shadows for depth without being heavy
  • Natural Animations: Spring-based animations with gentle easing
  • Responsive: Desktop-first but adapts to all screen sizes
